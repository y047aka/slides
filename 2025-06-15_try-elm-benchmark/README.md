# Elmのパフォーマンス、実際どうなの？ベンチマークに入門してみた

## 発表概要

Elmは型安全性や開発体験の良さが注目されがちですが、実際のパフォーマンスはどうなのでしょうか？
「Elmは遅いのか？速いのか？」という疑問に対して、実際にベンチマークを取り、得られた知見を共有します。
処理が重いサンプルコードを起点に少しずつ改良を進め、パフォーマンスを向上させてみましょう。
